# Home Assistant Add-on: YandexDNS

Automatically update your Yandex DNS IP address.

![Supports aarch64 Architecture][aarch64-shield] ![Supports amd64 Architecture][amd64-shield] ![Supports armhf Architecture][armhf-shield] ![Supports armv7 Architecture][armv7-shield] ![Supports i386 Architecture][i386-shield]

## About

[Yandex DNS][yandexdns] is a free service that points a DNS (sub-domains) to an IP of your choice.  You need to sign up for a Yadex DNS account before using this add-on.
